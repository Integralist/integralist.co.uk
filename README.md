# integralist.co.uk

Website built using [Go](https://golang.org/), [Hugo](https://gohugo.io/) and deployed via [Netlify](https://www.netlify.com/).

## New Post

- `hugo new ./posts/`

- `vim **<Tab>`(using [fzf](https://github.com/junegunn/fzf) for file searching)

- `hugo server` (for previewing)
